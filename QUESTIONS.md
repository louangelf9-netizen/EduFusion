# EduFusion — Complete Question Reference

> **Teacher / Developer Reference Document**
> This file documents every question in the `gameData` object from `script.js`.
> There are **4 game modes**, each with **10 levels** of **5 questions** each — **200 questions total**.

---

## Game Mode Descriptions

| Mode | Description |
|------|-------------|
| **📜 History Unlocked** | Multiple-choice questions about Philippine history, heroes, events, and culture. Players read a question and select the correct answer from 4 options. |
| **🖼️ Guess the Image** | Players are shown a photo and a text hint, then must identify the correct answer from 4 options. Covers heroes, landmarks, food, festivals, and national symbols. |
| **🔤 Spell It Right** | Players hear a Filipino word (text-to-speech) and must choose the correctly spelled version from 4 options. Focuses on Filipino vocabulary. |
| **🔠 Missing Letters** | Players see a word with vowels/letters replaced by underscores and must choose the complete correct word from 4 options. |
